# 🌲 ArbolABB — Árbol Binario de Búsqueda (BST)

Repositorio educativo que contiene una **implementación de un Árbol Binario de Búsqueda (ABB / BST)**, desarrollado principalmente en **Java**, con una estructura clara y pensada para el aprendizaje y la práctica de estructuras de datos.

Un Árbol Binario de Búsqueda permite organizar datos de forma jerárquica, facilitando operaciones como inserción y búsqueda de manera eficiente.

## 📌 ¿Qué es un Árbol Binario de Búsqueda?

Un **Árbol Binario de Búsqueda (ABB / BST)** es una estructura de datos jerárquica que cumple la siguiente propiedad:

- Los valores del subárbol izquierdo son menores que el nodo.
- Los valores del subárbol derecho son mayores que el nodo.

Gracias a esta característica, las operaciones de búsqueda e inserción son más eficientes que en estructuras lineales.

## 🚀 Contenido principal

### 💠 Implementación en Java

En la carpeta `Java-original` se encuentra una implementación base del ABB que incluye:

- Definición de nodos
- Inserción de valores
- Gestión de la raíz del árbol
- Clase de prueba para ejecución por consola

Este código sirve como base para ampliar funcionalidades como recorridos, eliminación de nodos o balanceo.
